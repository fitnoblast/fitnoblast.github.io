# FitNoBlast — Affiliate Landing Page

A single-page, SEO-ready, fully responsive landing page for FitNoBlast (health & fitness affiliate links).

## Files
- `index.html` — page markup, meta tags, structured data (JSON-LD)
- `style.css` — all styling, animations, responsive rules
- `logo.png` — **add your own logo file here** (square image, ideally 512×512px, transparent or dark background works best)
- `robots.txt` — allows all search engines to crawl the site
- `sitemap.xml` — basic sitemap for search engines

## Before you publish (important)
1. Drop your real **logo.png** into this same folder (same name, same folder as `index.html`).
2. Site is live at `https://fitnoblast.github.io/` — all SEO tags, sitemap, and robots.txt are already set to this domain.
3. If you're publishing with **GitHub Pages**:
   - Push these files to a repo (e.g. `fitnoblast-landing`)
   - Go to repo **Settings → Pages**
   - Set source to the `main` branch, root folder
   - Your site will be live at `https://<your-username>.github.io/<repo-name>/`

## What's included
- Two large, animated Amazon CTA buttons (India + USA) with hover shine and lift effects
- Social buttons for YouTube, Pinterest and Instagram
- Mobile-first responsive layout (works on all screen sizes)
- SEO meta tags: title, description, keywords, canonical, Open Graph, Twitter Card, JSON-LD Organization schema
- Accessible focus states and `prefers-reduced-motion` support

## Customizing
- Colors, fonts, and animation timing are all defined as CSS variables at the top of `style.css` under `:root`.
- Headline copy and section text live directly in `index.html`.
